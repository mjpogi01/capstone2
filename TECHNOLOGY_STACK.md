## Architecture

### Frontend
- **Framework**: React.js (SPA - Single Page Application)
- **Routing**: React Router DOM
- **State Management**: React Context API
- **HTTP Client**: Supabase Client (built-in fetch)
- **Styling**: CSS modules + Tailwind CSS

### Backend
- **Runtime**: Node.js
- **Framework**: Express.js
- **Architecture**: RESTful API
- **Server Port**: Configured via environment variables

### Database Architecture
- **Type**: Relational Database (PostgreSQL)
- **Hosting**: Supabase Cloud
- **Access**: 
  - Direct from frontend (Supabase client)
  - Via backend API (Express routes)
